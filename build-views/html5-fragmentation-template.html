HTML5-based platforms fragmentation

Key:

  • browser: re-use the API available in the browser
  • sysapp: re-use the API defined by SysApps
  • yes: has an API, but not the one available in browsers or defined by sysapps
  • alternative: supports the feature via an alternative mechanism, differently from the API available in browsers or defined by sysapps
  • no: feature not available
  • planned: planned feature, but not yet available
  • partial: partially available feature
  • unknown: unknown status for the feature

Thus, columns with yes or alternative are potential sign of fragmentation among HTML5-based platforms.
